 Open an existing file object. This method tries to open an existing file. If the file doesnt exist, an exception is thrown. The file will be opened with MPI file access. By using the openState variable, one can choose between opening the file in read-only mode or in read-write mode. Setting openState to PLI::HDF5::File::OpenState::ReadOnly will open the file in read-only mode. Setting openState to PLI::HDF5::File::OpenState::ReadWrite will open the file in read-write mode.
