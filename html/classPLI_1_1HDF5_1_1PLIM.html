Detailed Description

The PLIM class contains a few functions to make the integration of a generated HDF5 file easier with the Solr database used in the work group.

The class is a wrapper around the HDF5 library. It provides functions to add the necessary information to the HDF5 file to make it compatible with the Solr database.
