\hypertarget{md_README_autotoc_md9}{}\doxysection{Introduction}\label{md_README_autotoc_md9}
HDF5 is a widely used file format for storing and sharing scientific data. It is used in the Fiber Architecture Group to store brain data and pyramids of images. The pyramids allow easy visualization of the data with external tools.

However, using HDF5 can be troublesome when using C / C++. The library itself is written in C. While headers for C++ are provided, important features like MPI are not available.

To improve the usability of the library, this repository contains a C++ library wrapping features of HDF5 to make the usage in 3D-\/\+PLI projects easier.\hypertarget{md_README_autotoc_md10}{}\doxysection{Install instructions}\label{md_README_autotoc_md10}
\hypertarget{md_README_autotoc_md11}{}\doxysubsection{Requirements}\label{md_README_autotoc_md11}

\begin{DoxyItemize}
\item CMake 3.\+19 or newer
\item HDF5 with C and HL
\item MPI
\item C++17 compatible compiler
\end{DoxyItemize}
