\doxysubsection{Detailed Description}
Add attributes to an \mbox{\hyperlink{namespacePLI_1_1HDF5}{HDF5}} file, group, or dataset. 

This class allows to add, modify or delete attributes stored in an \mbox{\hyperlink{namespacePLI_1_1HDF5}{HDF5}} file. The class works on the pointers returned by the \mbox{\hyperlink{namespacePLI_1_1HDF5}{HDF5}} library. The pointers are available from the other classes of this library.
