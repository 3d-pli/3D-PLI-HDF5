Open an existing file object. 

This method tries to open an existing file. If the file doesnt exist, an exception is thrown. The file will be opened with MPI file access. By using the open\+State variable, one can choose between opening the file in read-\/only mode or in read-\/write mode. Setting open\+State to \mbox{\hyperlink{classPLI_1_1HDF5_1_1File_ad41a62cae2d952fe4648fd73585e85c5a131fb182a881796e7606ed6da27f1197}{PLI\+::\+HDF5\+::\+File\+::\+Open\+State\+::\+Read\+Only}} will open the file in read-\/only mode. Setting open\+State to \mbox{\hyperlink{classPLI_1_1HDF5_1_1File_ad41a62cae2d952fe4648fd73585e85c5a70a2a84088d405a2e3f1e3accaa16723}{PLI\+::\+HDF5\+::\+File\+::\+Open\+State\+::\+Read\+Write}} will open the file in read-\/write mode.
