\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPLI_1_1PLIM}{PLIM}} class contains a few functions to make the integration of a generated \mbox{\hyperlink{namespacePLI_1_1HDF5}{HDF5}} file easier with the Solr database used in the work group. 

The class is a wrapper around the \mbox{\hyperlink{namespacePLI_1_1HDF5}{HDF5}} library. It provides functions to add the necessary information to the \mbox{\hyperlink{namespacePLI_1_1HDF5}{HDF5}} file to make it compatible with the Solr database.
